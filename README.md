# Algorithms and Complexity (CS221)

This repository contains my laboratory activities and algorithm implementations for the **CS221 - Algorithms and Complexity** course. The focus of these exercises is on understanding algorithm design paradigms and analyzing their time and space complexity.

## Topics Covered

### Divide and Conquer
Efficiently solving problems by recursively breaking them into smaller subproblems.
- **Closest Pair of Points**: Finding the minimum distance between points in a 2D plane.
- **Convex Hull**: Finding the geometric boundary of a set of points.
- **QuickSort (Median of Three)**: Optimized sorting with balanced pivot selection.
- **Integer Exponentiation**: Large number calculations using binary exponentiation.
- **Majority Element**: Identifying the most frequent element in a set.

### Dynamic Programming
Solving optimization problems by combining solutions to overlapping subproblems.
- **Longest Common Subsequence (LCS)**: Sequence comparison and matching.
- **Optimal Binary Search Tree**: Cost-effective tree structure generation.
- **Rod Cutting**: Maximizing utility/profit from divided segments.
- **Subset Sum**: Identifying targets within a set of numbers.
- **Word Break**: Vocabulary-based string segmentation.
